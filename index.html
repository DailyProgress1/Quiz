HTML Quiz

1. What does HTML stand for?

  • Hyperlinks and Text Markup Language
  • Home Tool Markup Language
  • HyperText Markup Language
  • HighText Markup Language

2. Which tag is used to create a hyperlink in HTML?

  • <link>
  • <a>
  • <href>
  • <url>

3. What is the correct syntax for adding an image in HTML?

  • <img src="image.jpg" alt="Description">
  • <image href="image.jpg" description="Description">
  • <picture src="image.jpg" text="Description">
  • <img link="image.jpg" description="Description">

4. Which tag is used to create a numbered list?

  • <ul>
  • <ol>
  • <list>
  • <li>

5. How do you define the largest heading in HTML?

  • <h1>
  • <head>
  • <heading>
  • <h6>

6. Which tag is used for a line break?

  • <break>
  • <br>
  • <lb>
  • <ln>

7. What is the correct way to specify a background color in HTML?

  • <background="blue">
  • <body bg-color="blue">
  • <body style="background-color:blue;">
  • <bgcolor="blue">

8. Which tag is used to create a table?

  • <tab>
  • <table>
  • <td>
  • <tr>

9. How can you make a text bold?

  • <strong> or <b>
  • <em> or <i>
  • <bold> or <bt>
  • <thick> or <t>

10. Which tag is used to define metadata?

  • <meta>
  • <head>
  • <title>
  • <info>
